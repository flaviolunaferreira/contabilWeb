💡 Sistema de Ajuda

🏠 Dashboard

Sua visão geral financeira. Os cartões coloridos mostram: Saldo Realizado (dinheiro atual), Previsão de Saldo (estimativa futura), Despesas Previstas (contas a pagar), Total a Receber (dinheiro por vir), e Total Pago (gastos realizados).

➕ Adicionar Transações

Registre suas receitas (dinheiro que entra) e despesas (dinheiro que sai). Defina se a transação foi realizada (hoje) ou é uma previsão (futura). Use categorias para organizar melhor seus gastos.

📊 Previsões

Planeje seu futuro financeiro! Veja todas as transações futuras, filtre por período, categoria ou valor. Use os filtros para encontrar rapidamente o que procura.

📈 Gráficos

Visualize seus dados de forma intuitiva com gráficos de pizza, barras, linhas e áreas. Analise tendências, distribições e padrões dos seus gastos e receitas ao longo do tempo.

🔍 Dicas de Uso

  • • Clique nos cartões do Dashboard para ver detalhes das transações
  • • Clique nos cabeçalhos das tabelas para ordenar por aquela coluna
  • • Clique nas linhas das tabelas para editar transações
  • • Use ícones (?) para ajuda específica de cada campo
  • • Mantenha descrições detalhadas para melhor controle

💡 Dica: Passe o mouse sobre os ícones de ajuda (?) para dicas específicas!
